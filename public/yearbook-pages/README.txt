Flipbook pages — drop your Canva yearbook page exports here.

Export each page as PNG (or JPG/WebP) and name them in reading order:
  1.png, 2.png, 3.png ... (or 01.png, 02.png — sorting is numeric)

Page 1 becomes the front cover, the last page the back cover.
While this folder is empty the flipbook shows the generated student
spreads instead, so partial/missing exports never break the book.
