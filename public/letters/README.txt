Cut-out letter images for the hero title ("The Archive").

Drop transparent PNGs cropped from the Canva design here, one file per letter:

  t.png  h.png  e.png  a.png  r.png  c.png  i.png  v.png

Repeated letters can have distinct artwork via numbered variants — the 2nd
occurrence of a letter in the word tries "-2" first, the 3rd tries "-3":

  h-2.png   (the "h" in Archive, if different from the one in The)
  e-2.png   (the "e" in Archive)

Any letter without a file here automatically falls back to the CSS
paper-scrap style, so partial sets are fine.
